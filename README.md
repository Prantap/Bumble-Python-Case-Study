Bumble Profiles Data Analysis (Python Case Study)
Project Overview

This project analyzes a dataset of Bumble user profiles to uncover patterns in demographics, lifestyle preferences, income, and geographic distribution. The goal of this analysis is to demonstrate data cleaning, preprocessing, exploratory data analysis (EDA), and visualization techniques using Python. Through this project, we explore how user attributes such as age, height, income, diet, and location vary across the dataset and how these features relate to each other.

Objectives

Clean and preprocess real-world messy data
Handle missing values and outliers
Perform exploratory data analysis (EDA)
Engineer useful features from raw data
Identify patterns in user demographics and lifestyle
Visualize insights using Python libraries

Dataset Description
The dataset contains user profile information from Bumble including attributes such as:
Age
Height
Income
Diet preferences
Lifestyle habits
Geographic location
Other demographic information

This dataset is useful for understanding user behavior and demographic trends on dating platforms.

Project Workflow
1. Data Cleaning

The first step involved preparing the dataset for analysis by addressing common data quality issues.
Tasks performed:
1. Identified missing values
Calculated missing value percentages
Inspected column data types
Detected potential outliers
Visualized missing data patterns

2. Data Processing
After cleaning the dataset, several transformations were applied to make the data easier to analyze.
Steps included:
Binning numerical values into categories
Creating derived features
Performing unit conversions where necessary
These steps helped simplify the dataset and improve interpretability.

3. Exploratory Data Analysis
EDA was performed to uncover meaningful insights from the data.
Key analyses included:

i. Demographic Analysis
Understanding the distribution of users based on age and other demographic features.

ii. Correlation Analysis
Exploring relationships between numerical variables such as height, age, and income.

iii. Diet and Lifestyle Analysis
Examining patterns in user lifestyle preferences.

iv. Geographic Insights
Analyzing user distribution across locations.

v. Height Analysis
Understanding how user height varies across the dataset.

vi. Income Analysis
Studying income distribution and identifying trends.

4. Data Visualization
Visualizations were used to better understand patterns in the dataset.

Examples include:
Age distribution plots
Lifestyle preference charts
Correlation visualizations
Geographic insights
Visual analysis helps communicate insights more clearly and identify trends that may not be obvious from raw data.
